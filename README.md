# 🌏 Pangea

> 사람이 설명이 아니라 흔적으로 이해되는 세계를 만든다.

## 🚀 시작하기

```bash
pnpm install
pnpm dev
```

## 🛠 기술 스택

**Core**
- React
- TypeScript
- Next.js (App Router)

**Data & State**
- TanStack Query
- Supabase

**Styling & Animation**
- Tailwind CSS
- Motion.js

**Deployment**
- Vercel

## ⚙️ 개발 환경

- **Package Manager:** pnpm
- **Lint & Format:** Biome
- **Git Hooks:** Husky + lint-staged
- **Commit Convention:** commitlint
- **CI/CD:** GitHub Actions
